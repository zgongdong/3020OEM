Detailed Description

Sent in response to GattManagerCancelConnectAsCentral().

Status will be gatt_manager_status_success if a connection was cancelled. Other status codes represent a failure. gatt_manager_status_wrong_state indicates that there was no connection in progress. gatt_manager_status_failed may indicate that the cancel was tried too quickly.
