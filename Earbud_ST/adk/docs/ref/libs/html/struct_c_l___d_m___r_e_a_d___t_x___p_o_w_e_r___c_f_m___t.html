Detailed Description

Message containing the device's TX power for the particular connection.

A connection library client may request to read the local device's TX power for a particular connection. This message is returned in response to such a request. If the local device's TX power was read successfully then the status field should indicate this and the message will contain a valid TX power for that particular connection. The bluetooth address of the remote device which requested the read for the TX power is also returned.

Field Documentation

int8 tx_power

The local device TX power if the request was successful, otherwise invalid.
