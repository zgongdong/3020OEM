Detailed Description

Parameters for configuring transmission of broadcast data.

Field Documentation

uint16 interval_max

Maximum interval between CSB packets in slots. Range: 0x0002-0xFFFE; only even values are valid.

uint16 interval_min

Minimum interval between CSB packets in slots. Range: 0x0002-0xFFFE; only even values are valid.

bool lpo_allowed

Informs the BR/EDR controller whether it is allowed to sleep.

uint16 packet_type

Bitmask of permitted packet types.

uint16 supervision_timeout

Duration in slots after which the BR/EDR controller reports a CSB timeout event if it is unable to transmit a CSB packet. Range: 0x0002-0xFFFE; only even values are valid.
