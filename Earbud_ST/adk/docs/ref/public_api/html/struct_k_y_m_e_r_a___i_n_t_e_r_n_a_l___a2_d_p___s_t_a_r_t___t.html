The bits to clear in the client lock.

uint8 master_pre_start_delay

The number of times remaining the kymera module will resend this message to itself (having entered the locked KYMERA_STATE_A2DP_STARTING) state before proceeding to commence starting kymera. Starting will commence when received with value 0. Only applies to starting the master.
