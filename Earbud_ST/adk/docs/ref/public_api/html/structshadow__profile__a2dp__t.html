Field Documentation

l2ca_cid_t cid

The L2CAP cid of the shadowed A2DP media channel. The cid is used internally when primary to determine if an A2DP media channel is connected. When set to L2CA_CID_INVALID there is no A2DP media channel connected.

bool content_protection

Content protection is enabled/disabled
