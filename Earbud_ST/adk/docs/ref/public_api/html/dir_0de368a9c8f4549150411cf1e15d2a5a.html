temperature Directory Reference

Files

file  temperature.c
 Top level temperature sensing implementation. Uses a temperature sensor e.g. thermistor to actually perform the measurements.
 
file  temperature.h
 Clients of the temperature module should use this API. If a client just wants to read the current temperature, appTemperatureGet() should be used. If a client wants to be informed about changes in temperature with respect to limits, the client should register using #appTemperatureClientRegister. The module will then send status #temperatureMessages to the client.
 
file  temperature_config.h
 Configuration related definitions for the temperature module.
 
file  temperature_sensor.h
 Header file defining the temperature sensor API. Temperature sensors (e.g. thermistor) should provide an implementation for this API. The API is called by the temperature module.
