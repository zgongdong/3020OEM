Detailed Description

The role_selection internal structure for the client role.

This structure is visible to the application as it is responsible for managing resource to pass to the role_selection library. The elements of this structure are only modified by the role_selection library. The application SHOULD NOT modify the values at any time as it could lead to undefined behaviour.
