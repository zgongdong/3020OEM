Detailed Description

Contains status of the send Service Changed Indication request.

This message is received in response to GattServerSendServiceChangedIndication(). When an Indication is successfully sent, this message is returned with status 'gatt_status_success_sent'. Then, when the remote device responds to the notification, this message should be returned with status 'gatt_status_success'.

Parameters
