Detailed Description

The OBEX Connection parameters.

Field Documentation

bool auth

TRUE if OBEX auth is required as server.

TaskData connTaskData

TaskData for the connection context task. The connection context task returned by the ObexConnectRequest() and ObexConnectResponse() APIs will have the same TaskData in it.

uint16 sizeConnTask

Size required for the connection context task. The ObexConnectRequest() and ObexConnectResponse() API creates a connection context task of this size and returns that task by setting the connTaskData as its TaskData.

uint16 sizeTarget

Size of the target parameter string

const uint8* target

The Target header. The application must not free it during the entire OBEX session
