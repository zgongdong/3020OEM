uint32 frame_samples

The number of samples each frame represents once decoded. This parameter will not be used for SBC (where the frame header is used to calculate the number of samples or aptX. This parameter will need to be set for other codecs where there is more than one frame per packet (i.e. the extended audio header is being used).
