Detailed Description

BLE Low Duty Directed Advertising Parameters.

This structure contains the Direct Address to advertise through when the ble_adv_type is ble_adv_direct_ind_low_duty.

If NULL or address is empty then VM will Panic.

Field Documentation

uint16 adv_interval_max

Maximum advertising interval. Range: 0x0020..0x4000 Default: 0x0800 (1.28s)

uint16 adv_interval_min

Minimum advertising interval. Range: 0x0020..0x4000 Default: 0x0800 (1.28s)

bdaddr direct_addr

Public or random address to be connected too.

bool random_direct_address

FALSE for public remote address and TRUE for random address.
