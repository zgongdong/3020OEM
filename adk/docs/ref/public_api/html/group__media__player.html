Detailed Description

A component responsible for controlling audio (music) sources.

Responsibilities:

  • Media control (like play, pause, fast forward) - Commands are passed to the audio source media control API in response to UI inputs.

The Media Player uses Audio Domain Audio domain and UI UI domain.
