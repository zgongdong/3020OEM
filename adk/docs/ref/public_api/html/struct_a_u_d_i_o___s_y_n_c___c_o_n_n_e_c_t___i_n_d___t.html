Detailed Description

Request a change to "Connected" state.

Sent by the audio source to request that the audio sink does what is needed to get to the AUDIO_SYNC_STATE_CONNECTED state.

The audio sink must send a AUDIO_SYNC_CONNECT_RES in response, even if it cannot reach the requested state.
