Field Documentation

uint16* client_lock

If not NULL, set bits in client_lock_mask will be cleared in client_lock when the tone is stopped.

uint16 client_lock_mask

The mask of bits to clear in client_lock.

bool interruptible

If TRUE, the tone may be interrupted by another event before it is completed. If FALSE, the tone may not be interrupted by another event and will play to completion.

FILE_INDEX prompt

The prompt file index to play. FILE_NONE for tone.

promptFormat prompt_format

The prompt file format

uint32 rate

The tone/prompt sample rate

const ringtone_note* tone

Pointer to the ringtone structure to play, NULL for prompt.
