◆ appInitSetInitTask()

void appInitSetInitTask ( Task  init_task)

Set the Task to send initialisation complete messages to.

A utility function to give somewhere for app modules to store the init_task passed into the module initialisation function.

Only modules with asynchronous initialisation sequences may need to use this function.

Note: Only the init_task passed into the module initialisation function should be stored here.
